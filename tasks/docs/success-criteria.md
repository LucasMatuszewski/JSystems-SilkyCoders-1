# Success Criteria

## Functional

* 100% accuracy in rejecting returns > 30 days (date logic)
* 100% accuracy in rejecting complaints > 2 years (date logic)
* AI provides clear, understandable explanations for all decisions

## Technical

* Streaming responses with TTFT < 2 seconds
* Single JAR file deployment containing both frontend and backend
* Happy path works without errors (valid return and valid complaint flows)

## UX

* Intake form requires fewer than 5 initial inputs
* Mobile-responsive design works on common devices
* Users can understand rejection reasons and successfully resubmit
